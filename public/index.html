This is a Model Context Protocol (MCP) server providing comprehensive dice rolling capabilities for AI assistants.

Remote MCP Usage

Configure in Claude Desktop:

{
  "mcpServers": {
    "dice-rolling-remote": {
      "command": "npx",
      "args": ["@modelcontextprotocol/client-stdio", "connect", "<YOUR_DEPLOYMENT_URL>/sse"]
    }
  }
}

Available Tools

  • dice_roll - Roll dice using standard notation
  • dice_validate - Validate and explain dice notation

Example Notations

  • 1d20+5 - Basic roll with modifier
  • 2d20kh1 - Advantage (keep highest)
  • 2d20kl1 - Disadvantage (keep lowest)
  • 4d6kh3 - Character stats (keep best 3 of 4)
  • 3d6! - Exploding dice
  • 4d6r1 - Reroll 1s
  • 5d10>7 - Count successes (7+)

Important: For D&D 5e advantage, use 2d20kh1 (NOT 2d20)!

Protocol Endpoints

  • /sse - Server-Sent Events transport
  • /websocket - WebSocket transport
  • /http - HTTP transport
